
# üìä ML & Data Analysis: PM2.5 and Health Dataset for STEM Education

## üß† Overview

This project provides a practical, beginner-friendly example of how to apply machine learning and data analysis using Google Colab and Python in the context of real-world STEM learning. It is specially designed for **secondary school teachers and students**.

The project uses a simulated dataset on **PM2.5 air quality and related health symptoms**, demonstrating how data can be used to explore relationships and build simple predictive models.

---

## üéØ Objectives

* Empower STEM teachers and students to engage with real-world datasets
* Demonstrate step-by-step how to:

  * Load and explore datasets
  * Visualize trends and patterns
  * Train basic machine learning models (e.g. classification)
  * Interpret results and limitations

---

| `Simulated_PM2_5_and_Health_Symptoms_Dataset.csv` | Simulated dataset: PM2.5 levels and reported health symptoms                          |

---

## üöÄ How to Use

1. Open [Google Colab](https://colab.research.google.com/)
2. Upload the `.ipynb` file
3. Run the notebook cell-by-cell and follow the instructions
4. Upload the CSV dataset when prompted
5. Modify the code or try new experiments!

---

## üß™ Skills Developed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Correlation analysis
* Basic classification using scikit-learn
* Interpretation of confusion matrix, accuracy, and limitations of models
